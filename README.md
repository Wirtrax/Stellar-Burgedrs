# 🚀 Stellar Burgers

## 📌 About
Stellar Burgers is an interactive web application for a futuristic space-themed burger restaurant.  
The project includes real-time order tracking via WebSocket, responsive design for mobile and desktop devices, and a modern user interface.

## ✨ Features
- 🍔 Interactive burger constructor
- 📡 Real-time order feed (WebSocket)
- 📱 Fully responsive (mobile + desktop + touch support)
- 🔐 User authentication
- 🧪 End-to-end testing with Cypress
- ⚡ Fast build with Vite
